# 🎮 FiveM Dev Academy

Curso completo e **gamificado** para desenvolvimento de mods no FiveM — do zero ao avançado.

> Conteúdo baseado nas [documentações oficiais do FiveM](https://docs.fivem.net/docs/) e em tutoriais da comunidade no YouTube.

## ✨ Recursos

- 🗺️ **Roadmap** com 8 trilhas progressivas
- 🌙 Lua, ⚙️ FiveM Core (natives/eventos), 🖥️ NUI, 🗄️ Frameworks (ESX/QBCore)
- 🎨 Pré-curso de **Blender** (modelagem 3D para GTA V)
- 🚗 Veículos add-on, MLOs e roupas
- 🚀 Otimização, segurança e deploy de servidor
- 🧠 **Quizzes** interativos, ⚡ XP, níveis, 🔥 streak e 🏆 conquistas
- 💾 Progresso salvo localmente no navegador (localStorage)

## 🚀 Como usar

Acesse a versão hospedada no **GitHub Pages** ou abra o `index.html` localmente no navegador.

## 📁 Estrutura

```
├── index.html      # página principal
├── css/style.css   # tema dark gamer
└── js/
    ├── data.js     # conteúdo do curso (trilhas, aulas, quizzes)
    └── app.js      # gamificação e renderização
```
